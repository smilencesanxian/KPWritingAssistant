### 功能概述

### 给主 Agent 的建议

1. **词间距调整集成**：`justifyLine` 函数需要在 `drawAnswerContainer` 中调用，用于计算每行文字的词间距。最后一行应始终使用左对齐。

2. **渲染逻辑修改**：在 `drawAnswerContainer` 中渲染文字时，需要：
   - 判断是否为最后一行
   - 调用 `justifyLine` 计算额外间距
   - 使用 `doc.text()` 的 `wordSpacing` 选项应用间距

3. **边界条件注意**：
   - 单行长单词（超过行宽95%）会单独成行
   - 默写模式（mode !== 'tracing'）也需要应用相同的分行逻辑

4. **视觉验证**：PDF排版效果需要通过生成实际PDF文件人工检查，建议生成测试PDF对比调整前后的效果。

5. **后续优化建议**：
   - 考虑添加更多测试用例覆盖边界情况
   - 考虑性能测试，确保大文本处理不会过慢
   - 考虑不同字体大小下的排版效果
