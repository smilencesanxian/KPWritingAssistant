## 一、核心用户流程 (User Flows)

### 1.1 主流程：拍照批改 → 生成字帖

```

                                                        │
┌──────────┐     ┌──────────┐     ┌──────────┐         │
│  字帖预览  │←────│  生成字帖  │←────│ 批改结果页 │←────────┘
│          │     │  Loading │     │          │
└────┬─────┘     └──────────┘     └────┬─────┘
     │                                 │
     │    ┌──────────┐                 │
     └───→│ 下载/分享 │                 │
          │          │                 │
          └──────────┘                 │
                                       │
                    ┌──────────────────┘
                    │
                    ▼
          ┌──────────────────┐
          │  保存到亮点库/    │
          │  查看易错点专题   │
```

### 1.2 次要流程

#### 流程A：历史记录查看
```
首页 ──→ 点击"查看全部" ──→ 历史记录列表 ──→ 点击记录 ──→ 批改结果页（只读模式）
```

```
首页/批改页 ──→ 易错点Tab ──→ 易错点列表 ──→ 点击分类 ──→ 知识点详情 ──→ 生成练习纸
```

#### 流程C：亮点库管理
```
首页 ──→ 亮点库Tab ──→ 亮点列表 ──→ 查看详情/删除 ──→ 新建作文时提示复用
```

## 二、页面状态机 (Page State Machines)

### 2.1 拍照页状态机

```
                    ┌─────────────────────────────────────────┐
                    │                                         │
                    ▼                                         │
┌─────────┐    ┌─────────┐    ┌──────────┐    ┌──────────┐   │
│ 初始化   │───→│ 检查权限 │───→│ 相机预览  │───→│ 拍照完成  │───┤
│         │    │         │    │          │    │          │   │
└─────────┘    └────┬────┘    └──────────┘    └────┬─────┘   │
                    │                              │         │
                    │ 拒绝                         │ 确认     │
                    ▼                              ▼         │
              ┌──────────┐                   ┌──────────┐    │
              │ 权限引导  │                   │ OCR识别   │────┘
              │          │                   │ Loading  │
              └──────────┘                   └──────────┘
```

**状态说明**:
- `初始化`: 页面加载，准备相机
- `检查权限`: 检查相机/相册权限状态
- `权限引导`: 显示授权引导，提供设置跳转
- `相机预览`: 正常显示相机画面，可拍照/选相册
- `拍照完成`: 显示预览图，等待用户确认
- `OCR识别 Loading`: 进行中，显示进度动画

**状态转换触发**:
- 初始化 → 检查权限: 页面 mounted
- 检查权限 → 相机预览: 已授权
- 检查权限 → 权限引导: 未授权
- 权限引导 → 相机预览: 用户授权成功
- 相机预览 → 拍照完成: 用户拍照或选图
- 拍照完成 → 相机预览: 用户点击"重拍"
- 拍照完成 → OCR识别: 用户点击"确认"

### 2.2 批改结果页状态机

```
┌──────────────┐
│   加载中      │
│  (Fetching)  │
└──────┬───────┘
       │
       ▼
┌──────────────┐     ┌──────────────┐
│   加载成功    │────→│  范文级别选择 │
│  (Success)   │     │ (Level Select)│
└──────┬───────┘     └──────────────┘
       │
       │ 加载失败
       ▼
┌──────────────┐
│   错误状态    │
│   (Error)    │
└──────────────┘
```

**子状态 - 范文级别选择**:
```
用户点击级别 ──→ 显示Loading ──→ 重新生成范文 ──→ 更新范文区域
```

## 三、Loading状态详解

### 3.1 OCR识别 Loading

**触发条件**: 用户确认照片后
**结束条件**: OCR识别完成或失败
**显示内容**:
1. 动画图标（扫描动画）
2. 主标题："正在识别作文内容..."
3. 副标题（动态更新）:
   - "正在定位文字区域..."
   - "正在识别手写英文..."
   - "正在格式化文本..."
4. 进度条（模拟或真实进度）
5. 预计剩余时间

**取消操作**: 提供"取消"按钮，返回拍照页

### 3.2 AI批改 Loading

**触发条件**: OCR识别成功后
**结束条件**: 批改完成或失败
**显示内容**:
1. 动画图标（AI分析动画）
2. 主标题："AI正在批改作文..."
3. 副标题（轮播显示）:
   - "正在分析内容完整性..."
   - "正在检查语法错误..."
   - "正在评估词汇运用..."
   - "正在提取亮点..."
   - "正在生成优化范文..."
4. 预计时间：10-15秒

**技术提示**: 可接入 SSE (Server-Sent Events) 显示实时进度

### 3.3 字帖生成 Loading

**触发条件**: 点击"生成字帖"
**结束条件**: PDF生成完成
**显示内容**:
1. 动画图标（PDF生成动画）
2. 主标题："正在生成字帖..."
3. 副标题：
   - "正在排版答题卡格式..."
   - "正在渲染手写体字体..."
   - "正在生成PDF文件..."
4. 预计时间：2-3秒

## 四、错误处理策略

### 4.1 错误分级

| 级别 | 类型 | 处理方式 | 示例 |
|------|------|----------|------|
| L1 | 轻量错误 | Toast提示，自动消失 | 网络波动、操作频繁 |
| L2 | 中等错误 | Toast + 重试按钮 | OCR失败、批改失败 |
| L3 | 严重错误 | 页面级错误 + 引导操作 | 无网络、服务不可用 |
| L4 | 致命错误 | 弹窗阻断 + 退出引导 | 系统错误、数据损坏 |

### 4.2 具体错误场景处理

#### 网络错误 (No Network)

**检测**: 请求超时或 navigator.onLine = false
**处理**:
```
if (首次检测) {
  显示 Toast: "网络连接失败，请检查网络"
}

if (持续无网络) {
  页面级错误状态:
    - 插图: 断网插图
    - 标题: "网络连接失败"
    - 描述: "请检查网络设置后重试"
    - 按钮: [重新加载]
}
```

#### OCR识别失败

**可能原因**: 
- 图片模糊
- 手写难以辨认
- 非英文内容

**处理**:
```
Toast: "识别失败，部分文字难以辨认"

提供选项:
1. [重新拍摄] - 返回拍照页
2. [手动输入] - 进入文本编辑页，用户手动输入
```

#### AI批改失败

**可能原因**:
- 服务繁忙
- 请求超时
- 内容违规被拦截

**处理**:
```
Toast: "批改服务暂时繁忙，请稍后重试"

自动重试:
- 失败1次: 立即自动重试
- 失败2次: 提示用户手动重试
- 失败3次: 建议联系客服
```

#### 相机权限被拒绝

**处理**:
```
页面级引导:
  - 插图: 相机权限插图
  - 标题: "需要相机权限"
  - 描述: "请允许访问相机以拍摄作文"
  - 按钮1: [去设置] - 打开系统设置
  - 按钮2: [从相册选择] - 跳转相册选择
```

### 4.3 重试策略

| 操作 | 自动重试次数 | 重试间隔 | 降级方案 |
|------|-------------|----------|----------|
| OCR识别 | 1次 | 立即 | 手动输入 |
| AI批改 | 2次 | 3秒 | 联系客服 |
| PDF生成 | 2次 | 2秒 | 提示重试 |
| 数据加载 | 3次 | 2秒递增 | 缓存数据 |

## 五、数据状态管理

### 5.1 本地存储结构

```typescript
// 用户基础信息
interface UserInfo {
  openId: string;
  nickname: string;
  avatar: string;
  createdAt: string;
}

// 亮点库
interface Highlight {
  id: string;
  content: string;
  type: 'vocabulary' | 'sentence' | 'phrase';
  sourceEssayId: string;
  createdAt: string;
}

// 易错点
interface MistakePoint {
  id: string;
  category: string; // "主谓一致" | "时态" | "介词搭配"
  count: number;
  lastOccurAt: string;
  exampleErrors: string[];
  isMastered: boolean;
}

// 批改记录
interface CorrectionRecord {
  id: string;
  title: string;
  originalText: string;
  score: {
    total: number;
    content: number;
    communication: number;
    organization: number;
    language: number;
  };
  errors: Array<{
    position: [number, number];
    original: string;
    suggestion: string;
    type: string;
  }>;
  highlights: string[];
  modelEssay: {
    level: 'pass' | 'good' | 'excellent';
    content: string;
  };
  createdAt: string;
}

// 设置
interface UserSettings {
  weeklyReminder: boolean;
  reminderDay: number; // 1-7 周一到周日
  reminderTime: string; // "08:00"
  defaultFont: string;
}
```

### 5.2 数据同步策略

**MVP阶段**（本地存储优先）:
```
用户操作 ──→ 本地存储 ──→ 可选同步到云端（登录后）
```

**V2.0**（云端同步）:
```
用户操作 ──→ 本地存储 ──→ 立即同步到云端
          └─→ 离线队列（网络恢复后同步）
```

### 5.3 缓存策略

| 数据类型 | 缓存位置 | 过期时间 | 更新策略 |
|----------|----------|----------|----------|
| 用户基础信息 | LocalStorage | 永久 | 登录时更新 |
| 亮点库 | LocalStorage | 永久 | 实时更新 |
| 易错点 | LocalStorage | 永久 | 实时更新 |
| 批改记录列表 | LocalStorage + Memory | 1小时 | 下拉刷新 |
| 单条记录详情 | Memory | 会话级 | 重新加载 |
| 用户设置 | LocalStorage | 永久 | 实时更新 |

## 六、手势与交互细节

### 6.1 触摸手势

| 手势 | 触发区域 | 操作 | 反馈 |
|------|----------|------|------|
| 点击 | 全局 | 按钮/列表项 | 涟漪效果 |
| 长按 | 亮点库/历史记录 | 显示操作菜单 | 震动反馈 |
| 左滑 | 列表项 | 显示删除/编辑 | 弹性回弹 |
| 下拉 | 列表页 | 刷新数据 | 加载动画 |
| 上滑 | 批改结果页 | 查看更多内容 | 平滑滚动 |
| 双指缩放 | 字帖预览 | 放大PDF | 缩放动画 |

### 6.2 键盘交互

**文本输入场景**（手动输入作文）:
```
输入框获得焦点 ──→ 键盘弹出 ──→ 页面内容上移
                         └──→ 输入框保持可视区域

点击键盘外区域 ──→ 键盘收起 ──→ 页面恢复

点击"完成"按钮 ──→ 键盘收起 ──→ 提交输入
```

**键盘类型**:
- 英文作文输入: `type="text"` + `auto-capitalize="sentences"`
- 搜索: `type="search"`
- 数字: `type="number"` (如设置提醒时间)

### 6.3 物理按键

**Android返回键**:
```
首页 ──→ 提示"再按一次退出"
其他页 ──→ 返回上一页
弹窗打开 ──→ 关闭弹窗
```

## 七、性能与体验优化

### 7.1 图片处理

**拍照后处理**:
```
原始图片 ──→ 压缩 (最大 1920px 宽) ──→ 上传/识别
        └──→ 生成缩略图 (200px) ──→ 本地预览
```

**字帖PDF生成**:
```
服务端生成 ──→ 返回下载链接 ──→ 客户端缓存
```

### 7.2 骨架屏 (Skeleton Screen)

**适用页面**:
- 批改结果页（首次加载）
- 历史记录页
- 易错点详情页

**实现**:
```
灰色占位块 ──→ 脉冲动画 (shimmer) ──→ 数据加载完成替换
```

### 7.3 预加载策略

| 资源 | 预加载时机 | 方式 |
|------|-----------|------|
| 字体文件 | 小程序启动 | wx.loadFontFace |
| 常用图标 | 页面onLoad | 字体库已包含 |
| 用户历史记录 | 首页展示时 | 后台静默加载 |
| 范文模板 | 进入批改页 | 并行请求 |
