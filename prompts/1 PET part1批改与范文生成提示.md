## 二、批改要求

请严格按照以下PET官方评分标准进行批改。

### 四大评分维度（每维度0-5分）

这是考官评分的四个核心维度，也是批改时应该逐一检查的四个方面：

### 1. Content（内容）—— 是否“写了该写的”

| 分数  | 标准描述                         |
| ----- | -------------------------------- |
| 5分   | 完整覆盖所有要点，并补充相关细节 |
| 3-4分 | 覆盖大部分要点，可能有少量遗漏   |
| 1-2分 | 覆盖部分要点，遗漏较多或信息不足 |
| 0分   | 几乎没有覆盖要点或完全跑题       |

**批改检查点**：
- 是否回答了邮件中的所有问题？
- 是否有具体的细节支撑（如“上午好”是因为“人少”）？
- 是否因字数不足导致内容空洞？

### 2. Communicative Achievement（交际成就）—— 是否“写得像”

| 分数  | 标准描述                             |
| ----- | ------------------------------------ |
| 5分   | 格式规范，语气恰当，有清晰的交流意图 |
| 3-4分 | 格式基本正确，语气偶有不妥           |
| 1-2分 | 格式有误，语气生硬或不当             |
| 0分   | 完全不符合文体要求                   |

**批改检查点**：
- 邮件是否有恰当的称呼和结尾？
- 语气是否符合收件人身份（给老师要礼貌正式，给朋友可亲切自然）？
- 是否有“交流感”而非“列清单”？

### 3. Organisation（组织）—— 是否“有条理”

| 分数  | 标准描述                           |
| ----- | ---------------------------------- |
| 5分   | 分段清晰，逻辑连贯，恰当使用连接词 |
| 3-4分 | 有分段，基本连贯                   |
| 1-2分 | 结构混乱，连接词缺失或误用         |
| 0分   | 无结构，无法理解                   |

**批改检查点**：
- 是否合理分段（建议按要点分段）？
- 是否使用了连接词（and, but, so, because, firstly等）？
- 逻辑顺序是否清晰？

### 4. Language（语言）—— 是否“写得对、写得美”

| 分数  | 标准描述                         |
| ----- | -------------------------------- |
| 5分   | 词汇丰富，语法多样，错误极少     |
| 3-4分 | 词汇和语法基本正确，有一定多样性 |
| 1-2分 | 词汇贫乏，语法错误较多           |
| 0分   | 无法使用基本语言                 |

**批改检查点**：
- 是否混合使用了简单句和复合句（定语从句、状语从句等）？
- 词汇是否多样化（避免重复good, nice, happy）？
- 拼写、标点、大小写是否正确？

## 三、批改步骤

请按以下五步法进行批改，并在输出中体现每一步的检查结果：

### 第1步：内容审查
- 对照原题检查是否覆盖所有要点
- 检查是否有足够的细节支撑
- 确认字数是否在90-110词范围内

### 第2步：交际效果审查
- 检查称呼和落款是否恰当
- 判断语气是否符合收件人身份
- 评估整体是否有“交流感”而非“答题感”

### 第3步：组织结构审查
- 检查分段是否合理（每段一个中心意思）
- 标注连接词使用情况
- 评估逻辑流畅度

### 第4步：语言校对
- 逐句检查语法错误（时态、主谓一致、介词搭配）
- 检查拼写和标点（大小写、逗号、句号）
- 圈出重复词汇，建议更丰富的表达

### 第5步：总评与反馈
- 四个维度分别打分，计算总分（满分20分）
- 给出2-3条具体改进建议

## 四、亮点词句分析

请在批改中特别关注以下内容：

1. **指出亮点**：如果学生使用了生动、准确的表达（如“crack us up”、“delighted”等），请标注并说明为什么好。

2. **改造建议**：如果学生的表达过于平淡或重复，请给出具体的升级建议。例如：
   - 将“I like it”升级为“I'm really fond of it”或“It brings me joy”
   - 将“good”升级为“fantastic / amazing / wonderful”
   - 将简单句合并为复合句：“It was rainy. We stayed at home.” → “Because it was rainy, we stayed at home.”

## 五、常见失分点检查清单

请在批改时逐项确认，并在评语中说明是否存在以下问题：

- [ ] 字数不足（远低于100词）
- [ ] 语气不当（如给老师写信用“That's all”）
- [ ] 罗列要点无细节（只有“We need hats, water”而无理由）

** 范文生成要求 **
##### 1. 总体要求
1. 格式正确：包含称呼和结尾落款
2. 在保证作文质量的前提下，沿用用户亮点库中的好句好词，确保类似场景下的范文生成范围（结构、用语、亮点词句等）相对聚焦，以降低记忆量
3. 字数控制在98-110词左右

##### 2. 结构：严格遵循“三段式”
- **第一段**：回应来信 + 表明写信目的（2-3句）
- **第二段**：分点展开3个要点，每点“观点+理由/细节”（5-7句）
- **第三段**：结尾 + 互动提问（1-2句）

##### 3. 内容：每个要点都有“为什么”
| 低分表现                   | 高分表现                                                     |
| -------------------------- | ------------------------------------------------------------ |
| *I think summer is good.*  | *I think summer is the best time because the weather is warm and we can go swimming.* |
| *You should bring a coat.* | *You should bring a warm coat because it can get chilly in the evening.* |

##### 4. 语气：真实自然的交流感
- **给朋友**：使用 *Let’s… / How about… / Why don’t we… / I’m really excited!*
- **避免**：生硬的 *I am writing to inform you*（过于正式）或 *That’s all*（缺乏交流感）

##### 5. 语言：刻意使用1-2个复合句
- **条件句**：*If it rains, we can stay at home and watch a film.*
- **原因状语从句**：*I like this café because it’s quiet and the coffee is great.*
- **虽然句**：*Although I was tired, I still went to the party.*
